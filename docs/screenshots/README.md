# Screenshots

Drop PNGs here; the main README embeds three of them. Capture from a real
phone-width browser against your live instance (it looks far better than an
empty test instance).

Expected filenames (referenced by `../../README.md`):

| File | What to capture |
|------|-----------------|
| `sessions.png` | The session list with a few real sessions — name, folder, age, running/idle badges, the usage strip at top. |
| `create.png`   | The "new session" sheet open — parent dropdown + folder picker + name field. |
| `terminal.png` | The in-browser terminal modal attached to a live `claude` session. |

Optional extras you can add and embed wherever useful:

| File | What to capture |
|------|-----------------|
| `settings.png` | Settings sheet with the parent-folders list + directory browser. |
| `offline.png`  | The wake-offline-sessions sheet after a restart. |
| `selection.png`| Multiselect mode with the bulk-action bar. |

Tips:
- Phone-width (~390px) shows the mobile-first layout best.
- Mask anything private (real folder paths, project names) before publishing.
- Keep them reasonably small (PNG, ~30–80 KB each); the README scales to 30% width.
